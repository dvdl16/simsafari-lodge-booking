Jabulani Bookings Website

What is this website

The idea of this website is to have a single place to create and manage bookings for the guest houses on Jabulani. Users can view availability of guest houses and make bookings.

iNaturalist Integration

The Sightings page shows all recent species observations on Jabulani, as logged on iNaturalist.

Trails Map

The Trails Map page shows a map with recorded GPS tracks of all trails on Jabulani.

Why does the website look so strange/old/animated?

The theme was inspired by SimSafari, a Game Park construction and management simulation game released by Maxis on March 19, 1998. This game was played extensively by the website author years ago.

Technical Architecture overview

This website is serverless - the frontend is statically hosted on an S3 Bucket, and the backend runs on Lambda.

The frontend is built with Angular, a platform and framework for building client applications in HTML and TypeScript, and the backend has been programmed in Go.
